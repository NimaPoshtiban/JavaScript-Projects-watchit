# watchit!

## What is it?

A nodejs app for watching and running node files

### Dependencies

- Chalk
- lodash.debounce
- Chokidar
- Caporal

## Scripts

- <h3>watchit</h3>
<p>Runs index.js by default. if you want to run any other file just type the file name.
example: <b>watchit  test.js</b> </p>

- <h3>watchit -h</h3>
<p>shows other options</p>
